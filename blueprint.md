# '천원의 행복' 앱 (Happiness for 1000 Won)

## 1. 개요 (Overview)
어르신들의 따뜻한 한 끼 식사를 지원하는 1,000원 급식 서비스용 웹 애플리케이션입니다. 직관적이고 따뜻한 디자인을 통해 어르신들이 쉽고 즐겁게 서비스를 이용할 수 있도록 돕습니다.

## 2. 주요 기능 (Features)
- **메인 메뉴**: 8개의 직관적인 서비스 메뉴 버튼.
    - 오늘의 메뉴: 식단 확인 및 메뉴 선택.
    - 위치: 급식소 위치 안내.
    - 함께하는 사람들: 참여자 정보.
    - 후원하는 사람들: 후원자 명단.
    - 기부하기: 서비스 후원 기능.
    - 오늘식사 예약하기: 식사 예약.
    - 오늘식사 쏘기: 타인에게 식사 선물.
    - 식권 발행: 모바일 식권 생성.
- **메뉴 선택**: 맛있는 음식 사진과 함께 메뉴 확인 및 예약/선물 연동.

## 3. 디자인 가이드 (Design)
- **컬러**: #FF9800(주황), #FFF8E1(연노랑), #3E2723(진한 갈색).
- **폰트**: 크고 가독성 좋은 둥근 서체.
- **스타일**: 부드러운 그림자와 따뜻한 질감의 버튼.
